Maps

Polygon-based maps colored by dynamic data. Built in various platforms.

  • Leaflet (Open source client, free CloudMade tiles based on OSM)
  • Google Maps (2,500 free per day, then $4 / 1000 additional)
  • Open Layers (free, but clunky)
  • Microsoft (lacks native support for multi-polygons. tricky.)
  • Yahoo (under review.)
